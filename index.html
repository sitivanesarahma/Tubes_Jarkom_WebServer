Web server adalah bagian penting dari sebuah website yang bertugas memberikan layanan data. Sebagai pelayan permintaan pengguna, web server harus mampu beroperasi dengan stabil dan responsif agar pengguna dapat mengakses website dengan baik. Dalam tugas besar jaringan komputer, kami membangun web server menggunakan bahasa pemrograman Python dengan konsep socket programming dan library socket. Web server yang dibuat mampu menangani permintaan HTTP GET dan memberikan respon HTTP sesuai dengan permintaan yang diterima.
